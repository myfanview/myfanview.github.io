사용 방법

  • 자동 로드: 샘플 데이터(sampledata.json)가 자동으로 로드됩니다
  • 데이터 로드: 다른 JSON 데이터를 로드하려면 '데이터 로드' 버튼 클릭
  • 파일 업로드: 로컬 JSON 파일을 선택하여 업로드 가능
  • 센서 선택: 분석할 센서를 체크박스에서 선택 (같은 타입 여러 개 가능)
  • 다중 센서: 동일 타입의 여러 센서를 선택하면 모두 하나의 그래프에 표시됩니다
  • 그래프 타입: 다양한 신호처리 방식 선택
  • 선택 영역 신호처리: 시계열 그래프에서 영역을 드래그하면 하단에 신호처리 메뉴가 나타납니다
  • FFT: 주파수 도메인 분석 (회전 불균형, 베어링 결함)
  • STFT: 시간-주파수 2D 분석
  • Wavelet: 다중 스케일 분석
  • Hilbert: 신호 포락선 추출
